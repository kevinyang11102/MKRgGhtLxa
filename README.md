# 学生信息管理系统 python35

## 项目概述

学生信息管理系统是基于 Python Flask 框架，结合 MySQL 数据库、HTML、CSS 和 JS 技术进行设计与实现的。本项目提供完整的文档、PPT 和运行步骤，便于用户理解和部署。

## 技术栈

- 后端框架：Python Flask
- 数据库：MySQL
- 前端技术：HTML, CSS, JavaScript

## 功能模块

- 学生信息管理
- 学生信息综合分析

## 系统角色

- 管理员：负责系统的维护和数据处理
- 用户：进行学生信息的增删改查等操作

## 运行环境

- Python 环境推荐使用 Pycharm
- 数据库管理推荐使用 Navicat

## 部署步骤

1. 环境配置：安装 Python、Flask、MySQL 以及相关依赖
2. 数据库搭建：根据提供的 SQL 脚本建立数据库和表
3. 系统部署：运行 Flask 应用
